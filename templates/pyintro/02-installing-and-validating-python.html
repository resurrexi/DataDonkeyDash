  • Type python in console to start python CLI

Jupyter Notebooks¶

  • Start Menu > All Programs > Anaconda2 (64-bit or 32-bit depending on OS) > Jupyter Notebook
  • This will open a terminal window that will start the Jupyter Notebook
  • Once Jupyter starts, it will open a browser window with the Jupyter GUI for usage
  • Jupyter is primarily used for documenting and/or developing analyses
  • To close Jupyter, open the Jupyter terminal window, and press Ctrl-C
